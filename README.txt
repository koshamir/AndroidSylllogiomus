# Syllogimous Android PWA Clone

This is a mobile-friendly, noncommercial adaptation inspired by the original **Syllogimous v4** project by **4skinSkywalker**.

## Included
- installable PWA
- localStorage persistence
- tier progression
- arcade and playground modes
- history, stats, settings, export/import
- broad question set across the original taxonomy

## Android usage
1. Serve this folder with any static server.
2. Open it in Chrome on Android.
3. Use “Add to Home Screen” / “Install app”.

This package is a best-effort recreation designed for personal noncommercial use.
