Welcome to Syllogimous!

This mobile-friendly build is a noncommercial clone inspired by the original Syllogimous v4. It keeps the same overall flow: tiers, score, arcade/playground, question history, stats, local persistence, installable PWA behavior, and a broad set of logical reasoning exercises.

The original project presents Syllogimous as an Angular single-page logic training app with tiered progression, score-based advancement, local storage persistence, and grouped question generation across distinction, comparison, syllogism, arrangement, direction, graph matching, analogy, and binary problems. citeturn892759search0turn892759search2turn892759search3turn19file0turn20file0

This clone is packaged as a static PWA so it can be installed on Android from Chrome and run like an app.
